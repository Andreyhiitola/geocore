# GeoCore Academy — интерактивная платформа для оценки месторождений

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Three.js](https://img.shields.io/badge/Three.js-r128-green)](https://threejs.org/)
[![Version](https://img.shields.io/badge/version-1.2-blue)](https://github.com/Andreyhiitola/geocore/releases)

**GeoCore Academy** — это веб-приложение для геологов и горных инженеров, позволяющее визуализировать скважины, строить блочные модели и оценивать ресурсы месторождений прямо в браузере. Платформа имитирует работу профессионального ПО (Datamine Studio RM) и позволяет экспортировать результаты для дальнейшей работы.

## 🚀 Возможности

### 3D-визуализация
- **Скважины** — цветные линии с маркерами высоких содержаний (>5 г/т)
- **Каркас рудного тела** — золотистая сетка (формат OBJ)
- **Блочная модель** — цветные блоки внутри каркаса с градиентом от синего (низкое) до красного (высокое)
- **Управление камерой** — вращение, панорамирование, масштабирование

### Анализ данных
- **Загрузка CSV** — формат: `HoleID, From, To, Au_gpt` (или Cu_pct, Coal_m)
- **Загрузка OBJ** — каркас рудного тела (вершины и грани)
- **Интерполяция** — IDW, Ordinary Kriging (упрощённый), Nearest Neighbour
- **Классификация JORC** — Measured, Indicated, Inferred
- **Вариограмма** — экспериментальная по трём направлениям
- **Оценка ресурсов** — тоннаж, среднее содержание, металл

### Управление проектами
- **Сохранение проектов** — в localStorage браузера
- **Экспорт отчёта** — CSV блоков + текстовый отчёт
- **Экспорт для Datamine** — `.blockmodel`, `.wrl`, `.obj`, `.csv`

### Интерфейс
- **Три шаблона** — золото, медь, уголь
- **Тёмная/светлая тема**
- **Чат-бот** — консультации по горному делу
- **Блог** — статьи по методике и стандартам

🎮 Использование
1. Выбор шаблона
Нажмите на карточку месторождения (🥇 Золото, 🔴 Медь, ⚫ Уголь) — данные загрузятся автоматически.

2. Загрузка своих данных
CSV — выберите файл с колонками: HoleID, From, To, Au_gpt

OBJ — выберите файл каркаса рудного тела

3. Построение блочной модели
Настройте параметры (размер блока, метод, cutoff, стандарт)

Нажмите «Запустить построение блочной модели»

Дождитесь появления цветных блоков

4. Управление видимостью
Кнопка	Действие
🗻 Пустая порода	Скрывает/показывает блоки породы
🔮 Оболочка руды	Скрывает/показывает полупрозрачную оболочку
🕳️ Скважины	Скрывает/показывает линии скважин
🎥 Сброс камеры	Возвращает вид по умолчанию
5. Экспорт результатов
📦 .blockmodel — для Datamine Studio RM

🔷 .wrl — каркас для Datamine Studio Viewer

📐 .obj — универсальный формат

📊 .csv — блоки с координатами и содержанием

📥 Экспорт для Datamine
Форматы
Формат	Назначение	Открытие в Datamine
.blockmodel	Блочная модель	Studio RM → File → Open Block Model
.wrl	Каркас рудного тела	Studio Viewer → File → Open Wireframe
.obj	Каркас (универсальный)	Studio RM → Import → OBJ
.csv	Таблица блоков	Studio RM → File → Import CSV
Datamine Studio Viewer (бесплатный)
